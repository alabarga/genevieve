Administration

Genevieve is currently administered by Madeleine Ball. Administrators have access to Genevieve operations and private data. If Madeleine Ball chooses to add additional administrators to Genevieve's operations, the site will be updated to describe these.

Data use and privacy

Using the Open Humans API, Genevieve will have access to read your genetic data and Open Humans username.

Your genetic data will be used to generate a report. With the exception of data management within service providers used by this project (e.g. cloud services for the web server), individual private genetic data will not be shared with third parties. Individual report information generated from private genetic data will be private.

However if original data is publicly shared in Open Humans, this data and/or a resulting report may be publicly shared by Genevieve. Furthermore, other information (e.g. username) associated with the public genetic data in Open Humans may be associated with the public report shared by Genevieve.
